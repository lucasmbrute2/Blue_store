Blue Store

Nesse projeto a proposta é fazer um CRUD, escolhemos fazer uma aplicação para uma loja de eletronicos, onde temos um painel de administrador para ter acesso você precisa efetuar login, lá pode editar os itens a venda, adicionar um novo ou deletar os itens, também temos uma area de vendas onde o cliente pode selecionar itens e adicionar ao carrinho, usamos HTML, CSS e JS puro no frontend usamos o mini framework FLASK em python para o backend e fizemos o deploy com HEROKU.
